# gpui-table-core

Core traits and types for the gpui-table system.

## Overview

This crate provides the foundational abstractions for building tables in GPUI:

- **TableCell** - Trait for rendering values in table cells
- **TableRowMeta** - Trait defining table structure and metadata
- **TableRowStyle** - Trait for customizing cell and row rendering
- **TableLoader** - Trait for paginated data loading
- **Filter system** - Infrastructure for filtering table data

## Installation

```toml
[dependencies]
gpui-table-core = "0.5"
```

## Features

| Feature | Default | Description |
|---------|---------|-------------|
| `chrono` | Yes | DateTime type support |
| `rust_decimal` | No | Decimal type support |
| `fluent` | No | Internationalization |

## Core Traits

### TableCell

Implement this trait to render custom types in table cells:

```rust
use gpui_table_core::TableCell;

impl TableCell for MyType {
    fn render_cell(&self, cx: &WindowContext) -> impl IntoElement {
        div().child(self.to_string())
    }
}
```

### TableRowMeta

Defines the structure of a table row. Usually generated by `#[derive(GpuiTable)]`:

```rust
use gpui_table_core::TableRowMeta;

impl TableRowMeta for Product {
    const TABLE_ID: &str = "products";
    const TABLE_TITLE: &str = "Products";
    
    fn table_columns() -> Vec<TableColumn> { /* ... */ }
    fn cell_value(&self, col: usize) -> Option<Box<dyn TableCell>> { /* ... */ }
    fn table_filters() -> Vec<FilterConfig> { /* ... */ }
}
```

## Filter System

The filter module provides:

- `FilterConfig` - Filter metadata configuration
- `FilterType` - Supported filter types (Text, NumberRange, DateRange, Faceted)
- `Matchable<F>` - Trait for filtering rows against filter values
- `FilterEntitiesExt` - Extension methods for filter entity management

## Registry

When the `inventory` feature is enabled, table metadata is collected at compile time:

```rust
use gpui_table_core::registry::GpuiTableShape;

for shape in inventory::iter::<GpuiTableShape> {
    println!("Found table: {}", shape.struct_name);
}
```
